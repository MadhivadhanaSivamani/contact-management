The Contact Management System is a Python desktop application developed using the Tkinter library.
It allows users to add, edit, delete, and search contact information easily.
Contact details such as name, phone number, and email are stored in a JSON file.
The application automatically saves and loads contacts, ensuring data is not lost.
It features a modern purple and cyan user interface with an attractive design.
Animated background particles and button hover effects improve the user experience.
A live search feature helps users quickly find contacts from the list.
A contact counter displays the total number of saved contacts.
The project demonstrates Python concepts such as GUI development, file handling, event handling, and JSON data management.
This project is suitable for personal contact management and serves as a practical example of building interactive Python applications.
